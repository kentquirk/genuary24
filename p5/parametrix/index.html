Experimental Parameterized Sketcher

This is an experiment to see what would happen if I allowed all of the parameters of a graph to be controlled by functions that can be stacked. Each parameter has two functions that are applied one after the other along the stroke of the curve. These are shown on the left, with the boxes representing the values of that parameter along the curve. x and y control the position of the point along the curve; w is the width of the line at that point; r, g, b, and a are the red, green, blue, and alpha values of the color.

In animation mode (press 'a'), instead of applying the functions one after the other, the system smoothly interpolates between the values of the left and right functions.

Controls

  • right click in a function box -- cycle between function types: sin, cos, linear, constant, quadratic, cubic, bezier
  • left click and drag within a function box -- adjust the function parameters. Experiment!
  • r -- reset the sketch.
  • a -- toggle animation mode.
  • b -- toggle bounce or loop animation.
  • c -- copy the left column of functions to the right column
  • <, > -- increase or decrease animation speed.
  • 1-0 -- set animation duration to 1-10 seconds.
  • p -- pause the animation.
  • s -- save the current canvas to a png file.
