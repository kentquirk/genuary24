Random grid of colors

This creates an array of rectangles starting with one that covers the whole canvas, colored randomly.

Each round, it chooses a random rectangle and splits it into two rectangles either horizontally or vertically. Each of these new rectangles is then recolored with a variation of the previous color by choosing either r, g, or b from the parent rectangle. Then one half adds to that value (lightening it), and the other subtracts from it (darkening it). The other two components of the color are kept constant.

The colors fade from the starting value to the target value over time. When the grid gets too dense, it clears and starts over.

Controls

  • Press 'c' to clear the grid and restart.
  • Press space to pause and unpause the subdivision process.
  • Press 'l' to change the line colors between the rectangles between black, gray, white, and none.
