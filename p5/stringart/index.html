String Art

When I was a kid, I used to make string art by drawing an ellipse, drawing a circle around one of the foci, dividing the circle into a prime number of sections, then extending those sections onto the ellipse. I'd then hammer nails into the board and wrap string around every Nth nail. This is a digital version of that.

Controls

  • 1,2 -- set string width.
  • s -- choose a random set of 2-10 strings.
  • n -- set the number of nails.
  • c -- choose a random color palette for the strings.
  • b -- change the background color.
  • r -- restart.
