Experimental Parameterized Sketcher

This is an experiment to see what would happen if I allowed all of the parameters of a graph to be controlled by functions that can be stacked. Instead of setting the parameters of a line (position, width, color) to a constant, allow it to be adjusted mathematically along the path of the line.

Each parameter has two functions that are applied one after the other along the stroke of the curve. These are shown on the left, with the boxes representing the values of that parameter along the curve. x and y control the position of the point along the curve; w is the width of the line at that point; r, g, b, and a are the red, green, blue, and alpha values of the color.

Controls

  • right click in a function box -- cycle between function types: sin, cos, linear, constant, quadratic, cubic, bezier
  • left click and drag within a function box -- adjust the function parameters. Experiment!
  • r -- reset the sketch.
