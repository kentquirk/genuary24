Genuary 13 (Wobbly function day)

Seven Deadly Sines

This project combines seven sine wave generators at different frequencies to make one smoothly evolving image. It uses the basic concept of Lissajous figures but uses dots instead of continuous lines, and modifies colors and dot sizes dynamically too. Running demo here. Source code here.

Genuary 12 (lava lamp)

Not a Lavalamp

I wanted to create a lavalamp by doing a physics simulation of a mesh of points under various forces. I played with using a concave hull library to create the shape around the collection of points, and also with using a Voronoi diagram to create a mesh and then adding some weird spring-like forces to try to keep the mesh together. I had more fun playing with that and never bothered to finish making it look like a lavalamp. Running demo here. Source code here.

Genuary 8 (chaotic)

Kaotik

This is a simulation of rabbits and wolves in an ecosystem. The rabbits eat the grass, the wolves eat the rabbits, and the wolves die if they don't eat enough rabbits. There populations fluctuate chaotically. Running demo here. Source code here.

Genuary 4 (Pixels) & 6 (Screensaver)

Pixasplode

This loads a picture, draws it as circular pixels in random order, and then gives each pixel a weak velocity dependent on the hue of the pixel. It's a fairly pleasing way to blow up the image. Running demo here. Source code here.

Genuary 3 - The Droste Effect

Zooming in on "microbes"

The Droste Effect is when an image contains a smaller version of itself; in this case what I was after was that you're looking at microbes through a microscope, and you see them float onscreen. You move the slide to center one of them, zoom in, and find that the microbe contains more microbes.

It's not perfect but I like it.

Genuary 2 - No Palette

Random Grid of Colors

This creates an array of rectangles starting with one that covers the whole canvas, colored randomly. Each round, it chooses a random rectangle and splits it into two rectangles either horizontally or vertically. Each of these new rectangles is then recolored with a variation of the previous color by choosing either r, g, or b from the parent rectangle. Then one half adds to that value (lightening it), and the other subtracts from it (darkening it). The other two components of the color are kept constant. The colors fade from the starting value to the target value over time. When the grid gets too dense, it clears and starts over. Running demo here. Source code here.

Particles

This is a particle experiment. Each particle is thrown at a random angle, and when it hits the wall, it sticks. If it hits a stuck particle, it sticks. The particle chains grow in interesting ways. It clears and restarts whenever it gets full. Running demo here. Source code here.
